# tinyapi

Minimal FastAPI service exposing two endpoints for health checks and random number generation.

## Commands

```bash
make install                 # Install dependencies
make run                     # Run the server (port 8001)
make test                    # Run tests
make docker-build            # Build Docker image
make docker-run              # Build and run with Docker
make docker-run-indefinitely # Run in background
make docker-stop             # Stop container
```
